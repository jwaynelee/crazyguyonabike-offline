Notes:

There are three versions of each of your pictures: Large, small and thumbnail. This allows people with slow internet connections to select a faster way of accessing the website, because smaller pictures download faster. The "Large" size is the original version (see note on scaling below); "Small" is usually 50% of that, and "Thumbnail" is usually scaled to 10% of the original.

Scaling of large images: If your picture is very large (in terms of pixels rather than bytes) then it will be automatically scaled down to a more appropriate size for web display. This rule is only applied if the original image is more than 1000 pixels in both dimensions (height and width). So, if you'd like to maximize the size of your pics, then size them so that the smaller dimension is 1000 pixels or less. For example, a picture which is 1200 x 1100 will be automatically scaled down so that the smaller dimension is 1000, because both dimensions are more than 1000; but a pic that is 1400 x 1000 will be left as-is. Please note: Maximum allowed file size is 20.0 MB.
